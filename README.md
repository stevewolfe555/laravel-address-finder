# laravel-address-finder

## Summary

This package provides a facade for address searching. Currently the only driver is available is [Loqate](https://www.loqate.com). There is also a mock driver for testing that does not communicate with any API's

## Documentation

- Project docs: [`docs/README.md`](./docs/README.md)

## Tech Stack

- PHP

## Installation

```bash
composer require cyber-duck/laravel-address-finder
```

## Usage

Basic example:

```php
use CyberDuck\AddressFinder\Facades\Address;

$results = Address::search('10 Downing Street');
```

## Configuration

Example `.env`:

```dotenv
LOQATE_API_KEY=
ADDRESS_FINDER_CACHE=
ADDRESS_FINDER_CACHE_DRIVER=
ADDRESS_FINDER_DRIVER=
LOQATE_API_BASE_URI=
```

Required (depends on the configured driver):

- `LOQATE_API_KEY`

Optional:

- `ADDRESS_FINDER_CACHE`
- `ADDRESS_FINDER_CACHE_DRIVER`
- `ADDRESS_FINDER_DRIVER`
- `LOQATE_API_BASE_URI`
